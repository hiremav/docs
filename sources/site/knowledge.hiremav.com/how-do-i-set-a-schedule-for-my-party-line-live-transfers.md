# How do I set a schedule for my Party Line live transfers?

## Set a weekly schedule to tell Mav when you do (and when you don't) want calls

### How does it work?

Mav's Party Line Availability Schedule works by allowing you to set days and hours you're available for live call transfers.

Head over to the Party Line section inside the Mav dashboard and then click on "Availability." Once inside, you'll see a screen like the following:

![](https://knowledge.hiremav.com/hs-fs/hubfs/Screenshot%202024-01-23%20at%203-31-02%E2%80%AFPM-png-1.png?width=670&height=455&name=Screenshot%202024-01-23%20at%203-31-02%E2%80%AFPM-png-1.png)

On the left, you can select the days you are available. Additionally, you can set the start and end time _(or all day)_ for that day.

**💡 Note:** Agent availability must selected and turned on regardless of the schedule.
